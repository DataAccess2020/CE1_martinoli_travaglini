Introduction

In this little project we want to investigate how religion impacts the attitude towards euthanasia. Euthanasia, also know as mercy killing, is a legal,ethical and religious issue, which is reflected in a number of concepst and legislative solutions around the world. It is precisely on this last point, religion, that this paper wants to focus. The approach towards euthanasia by the main modern religions is certainly conflicting and usually reject this practice. In a letter written by Samaritanus bonus of the Vatican Congregation, it is reported that the Church believes it must reiterate as a definitive teaching that euthanasia is a crime against human life because, by such an act, a man chooses to directly cause the death of another innocent human. Unlike the approach of those in favour of this practice, euthanasia is directly compared to murder. Not only Christianity, but also Islam expressly forbids the use of this practice. Azza Hassan, of the National Center for Cancer Care and Research, Doha (Qatar), says that: Euthanasia, a practice accepted in the West countries, is not permissible in Muslim countries; on the contrary, it is viewed with great fear.

The dataset at our disposal helps us to see what the reference values of the people subjected to the questionnaire are, so we are going to relate a variable concerning religion and one about euthanasia. The purpose of this is to see if those who identify themselves as religious tend to reject the use of this practice, following the concepts of the religious institutions.

Our variables

In order to investigate the impact of religion on the attitude towards euthanasia, we choose to use the data from the European Values Study - wave 2017. The European Values Study is a research program on basic human values (for more information:European Values Study).

We decided to use the variable v6, which contains the answers to the question Q1 How important it is in your life: religion.

The values of the variable v6 are:

  • 1 = very important

  • 2 = Quite important

  • 3 = Not important

  • 4 = Not at all important

    • -2 = no answer
    • -1 = dont know

We decide to recode this variable into the variable imp_rel in which we set the values -2 and -1 as missing, and inverted the values of the original variable in order to have an increasing scale from 1 = Not at all important to 4 = very important.

Then, we choose as dependent variable the variable v156 which contains the answers to the question Q44H Please tell me whether you think it can always be justified, never be justified, or something in between: euthanasia. The variable contains values from 1 = never to 10 = always. We choose to recode also this variable so we can set the values -2 = no answer and -1 = dont know as missing; the other values remain unchanged.
